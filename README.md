# Exactum

## Visão Geral
O Exactum é um sistema de controle e previsão de estoque voltado para pequenos e médios comércios.
Seu objetivo é auxiliar na tomada de decisão sobre reposição de produtos, evitando tanto excesso
quanto falta de mercadorias.

## Problema
Pequenos comerciantes enfrentam dificuldades para equilibrar o estoque:
- Alguns produtos ficam parados por muito tempo
- Outros acabam rapidamente, gerando perda de vendas
- O controle manual torna a análise inviável à medida que o negócio cresce

## Solução
O Exactum centraliza o controle de entradas e saídas de produtos e, com base no histórico de vendas,
realiza previsões de quantidade necessária para um determinado período.

Exemplo:
> Para os próximos 7 dias, recomenda-se manter 50 unidades do produto "X".

## Público-Alvo
- Pequenos e médios comércios locais
- Lojas que ainda realizam controle manual ou pouco estruturado de estoque

## Fluxo de Uso Inicial
Na primeira versão:
1. O cadastro de produtos e movimentações é feito manualmente
2. O sistema registra entradas e saídas diariamente
3. Com base nesses dados, são geradas previsões de estoque por período

## Exemplo de Uso
Geraldo é dono de uma loja de artigos esportivos e enfrenta dificuldades para manter seu estoque equilibrado.
Enquanto alguns produtos vendem pouco, outros acabam rapidamente.

Utilizando o Exactum, Geraldo consegue:
- Identificar produtos com alta rotatividade
- Reduzir estoque parado
- Planejar compras com base em previsões mensais
